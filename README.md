# BOice Termin 2 Testprojekt

Dieses Projekt ist eine getrennte Testversion fuer den Frontend-MVP bis Termin 2.
Es liegt bewusst nicht im echten Projektordner `D:\BOice`.

## Inhalt

- Vue 3 mit TypeScript
- Campus-Feed mit Suche, Kategorien und Sortierung
- Formular zum Erstellen von Beitraegen
- Feed mit statischen Beispiel-Beitraegen
- Like/Dislike-Voting im Frontend
- Kommentar-Popup mit Account-Name und anonymem Toggle
- Frontend-Seiten fuer Anmelden, Registrieren, eigene Beitraege und Hinweise

## Stand fuer den 2. Praktikumstermin

- Frontend-MVP als Single Page Application
- Routen fuer Start, Beitraege, Anmelden, Registrieren, eigene Beitraege und Hinweise
- Vue Components fuer Navigation, Beitragsformular, Feed und Kommentare
- Responsive Layout fuer Desktop und mobile Ansicht
- Beispieldaten im Frontend, REST-API und Backend folgen spaeter

## Starten

```powershell
cd D:\Test\BOice-Termin2-Test
npm install
npm run dev
```

Danach im Browser oeffnen:

```text
http://localhost:5174
```
